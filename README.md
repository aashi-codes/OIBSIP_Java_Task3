## ATM Interface

### Objective
To design and implement a console-based ATM Interface using Java that simulates basic banking operations.

### Description
This project is a Java-based ATM system that allows users to register, log in securely, and perform essential banking operations such as withdrawal, deposit, transfer, balance inquiry, and viewing transaction history.  
The application is developed using Object-Oriented Programming principles and runs in a command-line environment.

### Features
- User Registration and Login Authentication
- Withdraw Money
- Deposit Money
- Transfer Money
- Check Account Balance
- View Transaction History

### Tools & Technologies Used
- Java
- IntelliJ IDEA
- Git & GitHub

### Project Structure
src/JavaProject/ATMInterface.java

### How to Run the Project
1. Clone the repository from GitHub
2. Open the project in IntelliJ IDEA
3. Ensure Java JDK is configured
4. Run the ATMInterface.java file
5. Follow the on-screen instructions in the console

### Learning Outcomes
- Understanding Java OOP concepts
- Handling user input using Scanner
- Implementing conditional logic and loops
- Managing transactions and data flow
- Using GitHub for project submission
